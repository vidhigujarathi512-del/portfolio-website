# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built using HTML5, CSS3, Flexbox, Grid, and JavaScript.

The website showcases my skills, projects, learning journey, achievements, and contact information through a clean and modern user interface.

---

## 🚀 Features

- Modern Hero Section
- Responsive Design
- Dark Mode Toggle
- CSS Grid & Flexbox Layouts
- Glassmorphism UI Design
- Interactive Hover Effects
- Project Showcase
- Timeline Section
- Skills & Tech Stack
- Contact Form
- Mobile-Friendly Design
